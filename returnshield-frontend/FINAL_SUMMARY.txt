💡 KEY DESIGN DECISIONS

  1. Component Library
     → Fewer, smarter components over many simple ones
     → Reduces duplicated code, ensures consistency

  2. Geist Design System
     → Geometric sans-serif for modern tech feel
     → Variable weights for hierarchy without size changes

  3. Teal + Amber + Dark Background
     → Distinctive fintech/healthcare aesthetic
     → Not a generic SaaS default

  4. Sidebar-First Layout
     → 248px persistent nav for context
     → Scales to mobile (hides on <890px)

  5. Bezel Cards (Layered Border)
     → Gradient outer, solid inner = depth effect
     → Differentiates from flat design trend

  6. Full TypeScript Coverage
     → Catch errors at compile time
     → IDE autocomplete for all props
